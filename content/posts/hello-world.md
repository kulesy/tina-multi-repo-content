# Hello Worlds

This is the first post in our multi-repo TinaCMS setup.

## What is Multi-Repo?

In a multi-repo setup:

* The **generator repository** contains the Tina schema and configuration
* The **content repository** (this one) contains the actual markdown files

## Benefits

* Separation of concerns between schema and content
* Content editors don't need to worry about schema changes
* Schema changes can be deployed independently
